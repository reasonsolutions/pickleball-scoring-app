# Nexa Font Integration

This directory should contain the Nexa font files from FontFabric.

## Required Font Files

Please download the following Nexa font files from https://www.fontfabric.com/fonts/nexa/ and place them in this directory:

1. **Nexa-Light.ttf** or **Nexa-Light.otf**
2. **Nexa-Regular.ttf** or **Nexa-Regular.otf** 
3. **Nexa-Bold.ttf** or **Nexa-Bold.otf**
4. **Nexa-Heavy.ttf** or **Nexa-Heavy.otf** (if available)

## Font Weights Available

- **Nexa Light** - For subtle text and secondary information
- **Nexa Regular** - For body text and standard content
- **Nexa Bold** - For headings and emphasis
- **Nexa Heavy** - For strong headings and branding

## Usage

Once the font files are added, they will be automatically loaded by the app and available through the font configuration in `src/utils/fonts.js`.

## Note

Nexa is a commercial font. Please ensure you have the proper license to use these fonts in your application.
